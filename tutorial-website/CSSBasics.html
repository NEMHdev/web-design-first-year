What is CSS?

CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media means that you can have an external file for all style features of the website.

Three Ways to Insert CSS There are three ways of inserting a style sheet:

  • External style sheet
  • Internal style sheet
  • Inline style

External Style Sheet

Allows you to change the look of an entire website by changing just one file. They can be written in any text editor and does not contain any html tags. The style sheet file is saved as a .css file.

Inline Styles

An inline style applied to a single element. To use inline styles, add the <style> attribute to the desired element. The style attribute can contain any CSS property.
