Html Basics:

All HTML documents must start with a type declaration called <DOCTYPE> which is not an HTML tag. This Instead it says what version of HTML you have used to write the page.

The HTML document itself begins with and ends with the html tag. It represents the root of an HTML document, which means that all the content is within this tag and contains all other Html tags. This is why DOCTYPE comes before the Html tags. The visible part of the HTML document is within the body tag.

General Tags

Tag Description
<!DOCTYPE> Determines the type of the document
<html> defines document as a html document
<body> holds the visible content
<h1> to <h6> html headings
<p> divides paragraphs
<br> Adds in a breakline
<hr> adds a line
<!-- --> used to comment out content
